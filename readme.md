#  Portfolio Personnel — Lydia El Gherissi

##  Description

Ce projet est un portfolio personnel développé en **HTML5, CSS3 et JavaScript** dans le cadre d’un projet de développement web.

L’objectif était de créer un site moderne, responsive et interactif permettant de présenter :

* mon parcours,
* mes compétences,
* mes projets,
* et mon identité visuelle.

Le site a été entièrement codé sans framework afin de maîtriser les bases du développement front-end.

---

#  Inspiration & Direction Artistique

Le design du portfolio s’inspire :

* des univers éditoriaux,
* des collages artistiques,
* des campagnes mode,
* et d’ambiances estivales chaleureuses proches de l’univers de Zara Larsson.

La palette repose principalement sur :

* des tons rosés,
* orangés,
* dorés,
* avec une atmosphère lumineuse et créative.

L’objectif était de créer un portfolio à la fois :

* artistique,
* immersif,
* moderne,
* tout en gardant une bonne lisibilité.

---

#  Technologies utilisées

* HTML5

---

#  Responsive Design

Le site est entièrement responsive et s’adapte :

* aux ordinateurs,
* aux tablettes,
* et aux téléphones mobiles.

Les media queries CSS permettent d’adapter :

* les grilles,
* les espacements,
* et la navigation selon la taille de l’écran.

---

#  Fonctionnalités

## Navigation

* Navigation fluide entre les sections
* Header fixe

## Animations

* Apparition progressive des sections
* Effets hover
* Transitions CSS

## Slider

* Slider interactif dans la section “À propos”

## Modal projets

* Ouverture dynamique des projets
* Affichage des descriptions avec JavaScript

1. Télécharger le projet
2. Ouvrir le fichier `index.html`
3. Le site se lance directement dans le navigateur
